Detailed Description

Helper class used by IPOPT to solve a NLP. Used here to implement a MPC controller We define a state space system (A,B), a LQR cost defined by (Q,R) with terminal cost P, we set the initial condition to x0, the horizon to N, and enforce polytopic constraints on state Fx * x(t) <= bx and polytopic constraints on input Fu * u(t) <= bu
